  • macOS 15.0 (Sequoia) or later
  • Apple Silicon or Intel Mac
  • An AI provider API key (OpenAI, Anthropic, Google, or xAI) — or Ollama for free local AI

Frequently Asked Questions

How do I set up an AI provider?

Open the app, go to Settings, select your preferred AI provider, and enter your API key. Keys are stored securely in your macOS Keychain.

Can I use the app without an API key?

Yes! Install Ollama for free local AI processing. Run brew install ollama then ollama pull llama3.2-vision:11b. The app will auto-detect it.

Is my data safe?

Yes. Files are processed locally on your Mac. When using a cloud AI provider, only file metadata and extracted text are sent to the provider's API — never through our servers. With Ollama, nothing leaves your Mac at all. See our Privacy Policy.

Can I undo file renames?

Yes. Every operation is logged and reversible. Go to Recent Actions in the app to undo any rename or move.

What file types are supported?

The app works with all common file types including PDFs, images (JPEG, PNG, HEIC), documents (DOCX, TXT), and more. Vision analysis works with images and scanned PDFs.

The app isn't detecting Ollama

Make sure Ollama is running (ollama serve) and that you've enabled it in the app's Settings. The app only checks for Ollama if you've toggled it on.

I'm getting an API error

Check that your API key is valid and has sufficient credits/quota. Try regenerating your key from the provider's dashboard and re-entering it in Settings.

How do I request a refund?

Refunds are handled by Apple. Visit reportaproblem.apple.com to request a refund.
